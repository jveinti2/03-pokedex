# Ejecutar en desarrollo

1. clonar repositorio
2. ejecutar `npm i`
3. Tener CLI nest instalado `npm i -g @nest/cli`
4. Levantar la base de datos `docker-compose up -d`
5. Reconstruir bd con `http://localhost:300/api/v2/seed`

# Stack

- MongoDB
